EP-ui

사용법은 아래 사용법을 복사해서 사용하시면됩니다.
사용법 설명
copy demo dialog 
PubEPUI.dialog.frame('./pubEPUI.html', {
							title : 'pubjs'
							,width : '480'
							,height : '355'
							,directCall :true
						});
 옵션은 jquery-ui-1.10.3.custom.min.js dialog 동일합니다.
copy demo toast 
PubEPUI.toast.view({
						heading: 'Information'
						,text: '테스트'
						,icon:'info'
					})
 옵션은 jquery.toast 동일합니다.
copy demo modal 
PubEPUI.modal.open({
		message : '메시지'
		,width:  '400px'
		,height : '200px'
		,okCallback :  function (){
		alert('1')
}
		,okBtn : '확인'
		
	});
 옵션은 jquery.toast 동일합니다.
